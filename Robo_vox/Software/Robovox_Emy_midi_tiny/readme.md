# MIDI controls
- MIDI note ON/OFF mapped to the SC02's phonemes
- Velocity
- Pitch
- Mod Wheel mapped to filter
- CC 64 sustain ON/OFF to choose between internal and external carrier

The MIDI clock is available on the Busy jack
